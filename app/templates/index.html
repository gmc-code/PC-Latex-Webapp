Welcome to the GMC LaTeX worksheet generator.

Use the menu bar to navigate to the desired worksheet.

Pdf files, or zip files containing pdfs, will take several seconds to be generated and downloaded to your downloads folder.

As a rough guide, a progress bar will be shown, which takes about 7 seconds to animate.

Some graphics intensive pdfs will take longer; wait for the page to stop animating in the browser tab.

Automatic file removal:

Due to hosting limitations, generated pdfs will be removed automatically.

In the Admin menu, Clean Output will remove all generated files.
